Lenguaje Java

JavaScript (JS) es un lenguaje de programación de alto nivel, interpretado y dinámico, fundamental para el desarrollo web. Se utiliza principalmente para crear sitios web interactivos, dinámicos y funcionales, ejecutándose en el navegador del cliente (lado del cliente), lo que permite animaciones, validación de formularios y actualizaciones de contenido en tiempo real sin recargar la página.

Lenguaje Python

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, famoso por su sintaxis clara y legible, que facilita la escritura de código. Creado por Guido van Rossum en 1991, es ampliamente utilizado en ciencia de datos, inteligencia artificial, desarrollo web y automatización.

Administrador Base de Datos

MySQL es un sistema de gestión de bases de datos relacionales (RDBMS) de código abierto, basado en el lenguaje SQL (Structured Query Language). Permite organizar, almacenar y consultar datos en tablas de filas y columnas, destacando por ser rápido, confiable y escalable, siendo ideal para aplicaciones web, comercio electrónico y sitios de alto tráfico.
